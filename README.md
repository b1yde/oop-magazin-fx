# Магазин — расчётное задание по ООП

Имитация процесса обслуживания покупателей в магазине с несколькими
отделами и кассами. Время дискретно. JavaFX-визуализация: длины очередей
касс, выручка по отделам, продажи по товарам, доли видов товара.

Студент: **Исаев А.Л.**, группа **ПИЭ-52**, АлтГТУ им. И.И. Ползунова,
факультет ИТ, кафедра ИСЭ, Барнаул 2026.

## Структура

- `Simulyaciya.java` — ядро имитации (шаги времени, генерация покупателей,
  завоз товара, списание просрочки, отчёты).
- `MagazinFxApp.java` — главное окно JavaFX (диаграммы, кнопки, отчёты).
- `ObrabotchikKnopki.java` — обработчик кнопок (реализует `EventHandler`).
- `Otdel.java`, `Kassa.java` — отдел и касса со своим складом / очередью.
- `Pokupatel.java`, `Korzina.java`, `StrokaKorziny.java` — покупатель и корзина.
- `Tovar.java`, `VidTovara.java`, `Partiya.java` — товар, вид товара, партия.
- `Диаграмма.png` — итоговая диаграмма классов UML.

## Сборка и запуск

Нужны JDK 17+ и JavaFX SDK 21. Скачайте JavaFX SDK с
[gluonhq.com/products/javafx](https://gluonhq.com/products/javafx/) и распакуйте,
затем (Windows, PowerShell):

```powershell
$JFX = "C:\путь\к\javafx-sdk-21\lib"
javac --module-path $JFX --add-modules javafx.controls *.java -d out
java  --module-path $JFX --add-modules javafx.controls -cp out MagazinFxApp
```

## Возможности интерфейса

- Шаг имитации / прогон N шагов.
- Продажи и остатки по выбранному товару или виду товара.
- Скоропортящиеся товары с остатком срока ≤ N шагов.
- Сезонные товары с нестандартной ценой.
- Среднее число покупателей и средняя сумма покупки за шаг.
- Покупатели, не дождавшиеся обслуживания.
- Выручка по кассам, отделам и магазину в целом.
- Диаграммы: длины очередей, выручка по отделам, продажи по товарам,
  доли видов товара (PieChart).
